# KIN Home Design System

## Colors
```
--kin-orange: #f97316        (primary CTA, accent)
--kin-orange-light: #fb923c  (hover states)
--kin-orange-dark: #ea580c   (pressed states)
--kin-dark: #0a0a0a          (backgrounds)
--kin-dark-2: #111111        (cards, elevated surfaces)
--kin-dark-3: #1a1a1a        (hover backgrounds)
--kin-cream: #ece6e4         (light text on dark, section backgrounds)
--kin-sage: #8c968d          (secondary accent, subtle badges)
--kin-blue: #0050bd          (links, secondary CTAs)
--text-primary: #fafafa      (headings on dark)
--text-secondary: #a3a3a3    (body text on dark)
--text-muted: #525252        (captions, labels)
--border: #262626            (card borders, dividers)
```

## Typography
- Font: Poppins (300 light, 400 regular, 500 medium, 600 semi, 700 bold)
- h1: 48-64px / 700 / tight tracking
- h2: 36-48px / 600
- h3: 24-30px / 600
- body: 16-18px / 400 / relaxed line height
- small: 14px / 400

## Spacing
- Section padding: py-20 md:py-28
- Container: max-w-7xl mx-auto px-6
- Card padding: p-6 md:p-8
- Component gaps: gap-4 to gap-8
